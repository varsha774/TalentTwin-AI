# 🚀 TalentTwin AI

## AI-Powered Resume Analyzer & Career Recommendation System

TalentTwin AI is an AI-based career assistant that helps students and job seekers analyze their resumes, identify skills, calculate resume scores, and get suitable career recommendations.

The platform uses resume parsing and intelligent skill matching to provide insights for improving career opportunities.

---

## ✨ Features

- 📄 Upload Resume PDF
- 🔍 Extract Resume Content Automatically
- 🧠 AI-Based Resume Analysis
- 💯 Resume Score Calculation
- 🛠️ Skill Detection
- 💼 Job Role Recommendation
- 📊 ATS Optimization Suggestions
- 🎯 Career Improvement Guidance

---

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/varsha774/TalentTwin-AI.git
```

Go to project folder:

```bash
cd TalentTwin-AI
```

---

# Backend Setup

Navigate to backend:

```bash
cd backend
```

Install dependencies:

```bash
npm install
```

Create `.env` file:

```
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

Start backend:

```bash
npm run dev
```

Backend runs on:

```
http://localhost:5000
```

---

# Frontend Setup

Open another terminal:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start frontend:

```bash
npm run dev
```

Frontend runs on:

```
http://localhost:5173
```
